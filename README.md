井字棋 (Tic-Tac-Toe)

一个基于控制台的井字棋游戏，由 C++ 编写。玩家执黑（1），电脑执白（2），采用 Minimax 算法实现，电脑为不可战胜的难度。

游戏说明

玩家: 您执黑棋，用 1 表示。

电脑: 电脑执白棋，用 2 表示。

棋盘: 游戏在一个 3x3 的棋盘上进行。

操作: 轮到您时，根据提示输入您想落子的坐标 x 和 y (范围均为 1-3)，用空格隔开。

游戏规则

玩家与电脑轮流在 3x3 的棋盘上落子。

率先将自己的棋子（1 或 2）在任意一行、一列或一条对角线上连成三子者获胜。

如果棋盘被填满而没有任何一方获胜，则游戏为平局。

实现细节

核心算法: 电脑 AI 使用 Minimax 算法 进行决策。该算法通过递归地模拟所有可能的后续走法，为自己选择最优解，确保电脑在任何情况下都不会输。

游戏状态:

0: 空格

1: 玩家（黑棋）

2: 电脑（白棋）

胜负判定: 程序会实时检测棋盘状态，判断是否有玩家达成胜利条件或棋盘已满形成平局。
